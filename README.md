# svgmodule

_work in progress_

This JavaScript library provides a set of [SVG](https://www.w3.org/TR/svg) DOM assemblers.

## Installation

```
npm install svgmodule
```

## Development

```
git clone git@github.com:aristov/svgmodule.git
cd svgmodule
npm install
npm run watch
```
